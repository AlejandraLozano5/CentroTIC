Subistema estación de monitoreo ambiental

El subsistema de estación monitoreo ambiental se encarga de ir recolectando variables ambientales como temperatura, humedad relativa, presión atmosférica, luz ultravioleta, precipitaciones, radiación solar, y velocidad y dirección del viento, estas variables se recolectarán cada vez que se haga una medida del espectro radioeléctrico por el subsistema RFI con el fin de establecer correlaciones que permitan estudiar fenómenos ambientales con medidas de RF y así establecer eventos que puedan afectar las medidas del espectro, este subsistema se representa en la figura donde se compone principalmente por:

  • Estación metereológica General Purpose Weather Station metone
  • Sistema embebido raspberry pi4 para comunicación con el servidor
  • Comunicación vía Ethernet usando cable CAT6a
  • Conexión eléctrica junto con núcleos de ferrita para reducir la circulación de corrientes parásitas que puedan generar una onda electromagnética
